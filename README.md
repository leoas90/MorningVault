# MorningVault

Privacy-first iOS morning briefing app. HealthKit, EventKit, WeatherKit, RSS, Foundation Models — all on-device, no tracking.

- **Bundle ID:** `com.yeziddr.morningvault.dev`
- **Min iOS:** 26.1
- **Build:** open `MorningVault.xcodeproj` in Xcode 16.4+
- **CI:** `.github/workflows/build.yml` (push to `main` → build + TestFlight upload)
- **Backend:** `backend/` — FastAPI proxy wrapping Polygon.io for market data
